# Antigravity 연동 수정 계획

작성일: 2026-03-19
기반 문서: `.research/research_260319_v12.md`

## 목표

`ollama-proxy`를 통해 antigravity 모델을 호출할 때 다음 두 문제를 해결합니다.

1. upstream이 지원하지 않는 antigravity 모델 ID가 노출되어 400이 나는 문제
2. upstream이 이미 처리한 구조화된 429/503을 다시 재시도해서 응답이 과도하게 늦어지는 문제

## 구현 계획

1. antigravity 모델 정규화 계층 추가 - 완료(Done)
   - `src/handlers/chat.py`에 antigravity 전용 모델 alias 정규화 로직을 추가합니다.
   - `antigravity:claude-*`, `antigravity:gemini-*` 같은 legacy 또는 plain 모델명을 upstream 실제 공개명으로 변환합니다.
   - 이미 올바른 `anti-*`, `gcli-*` 모델은 그대로 유지합니다.

2. antigravity upstream 에러 passthrough 추가 - 완료(Done)
   - `src/providers/base.py`에 "구조화된 upstream 에러는 재시도하지 않고 즉시 반환"하는 경로를 추가합니다.
   - 최소 범위는 antigravity provider에 한정합니다.
   - `model_capacity_exhausted` 같은 명시적 에러 코드는 재시도 대상에서 제외합니다.

3. 라우트 응답 처리 보강 - 완료(Done)
   - `src/routes/openai.py`와 `src/routes/ollama.py`에서 passthrough된 upstream 에러를 적절한 상태 코드로 반환하도록 맞춥니다.
   - OpenAI 경로는 원본 에러 구조를 최대한 유지합니다.
   - Ollama 경로는 기존 Ollama 오류 포맷으로 안전하게 변환합니다.

4. 설정 설명 정리 - 완료(Done)
   - `config.py`의 antigravity 관련 오래된 주석을 현재 동작과 맞게 수정합니다.

5. 회귀 테스트 추가 - 완료(Done)
   - 지원되지 않던 plain antigravity 모델이 정상 alias 변환되는지 테스트합니다.
   - antigravity가 503 `model_capacity_exhausted`를 반환할 때 외부 재시도가 발생하지 않는지 테스트합니다.
   - antigravity가 400 `Unknown model`을 반환할 때 즉시 에러가 전달되는지 테스트합니다.

## 검증 계획

1. 단위 테스트 실행
   - antigravity alias 정규화 테스트
   - passthrough/no-retry 테스트

2. 로컬 실제 호출 검증
   - `antigravity:anti-claude-sonnet-4-6`
   - `antigravity:gcli-gemini-3-pro-preview`
   - legacy alias 예시 1종

3. 로그 확인
   - 503 또는 400 발생 시 `ollama-proxy`가 같은 요청을 반복 재시도하지 않는지 확인

## 검증 결과

- `python3 -m unittest tests.test_chat_handler_limits tests.test_provider_error_handling` 통과
- `python3 -m compileall src tests config.py` 통과
- 로컬 컨테이너 재빌드 후 실호출 검증 완료
- `antigravity:claude-sonnet-4-6` -> alias 정규화 후 `200`
- `antigravity:gcli-gemini-3-pro-preview` -> `200`

## 완료 기준

- `ollama-proxy`가 노출하는 antigravity 모델명 중 legacy plain 이름도 실제 upstream 호출로 연결됩니다.
- antigravity upstream의 구조화된 400/429/503은 즉시 사용자에게 반환됩니다.
- 동일 요청이 `ollama-proxy`에서 중복 재시도되어 장시간 대기하는 현상이 사라집니다.
